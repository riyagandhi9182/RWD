Networking Basics

Question 1

How long is an IPv6 address?

A. 32 bits

B. 128 bytes

C. 64 bits

D. 128 bits

Answer

Option D

Explaination

An IPv6 address is 128 bits long.

Question 2

Which class of IP address has the most host addresses available by default?

A. A

B. B

C. C

D. A and B

Answer

Option A

Explaination

Class A addressing provides 24 bits for host addressing.
